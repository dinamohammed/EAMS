Material Internal Requisitions by Employees/Users

Material Internal Requisitions apps help you to manage product internal requisition processes by employees and users. After installing this odoo module you see the option to assign stock location for each users and employees, once internal requisition placed for required goods and material, by default it will be placed from the configured location for users and employee.Internal requisition comes from the complete flow of the confirm order and approve order with department and manager approval workflow which makes internal picking for the warehouse stock movement etc.

Benifits and Features

Features

- Allow your employees to Create Material Internal Requisition.
- Employees can see only him/her records of Product/Material Internal Requisitions.
- Employees can request multiple materials/items on single Material Internal Requisition Request.
- Department Manager can Approve of Department Approval of Material Internal Requsition.
- Department Manager can also Reject Material Internal Requsition.
- Requisition User can Approve Material Internal Requisition.
- Requisition User can also Reject Material Internal Requsition
- Email notifications to Department Manager, Requisition Manager for approval.
- Request for Material Internal Requisition will go to stock/warehouse as internal picking / internal order and purchase order.
- Warehouse can dispatch material to employee location and if material not present then procurment will created by Odoo standard.
- Material Internal Requisition user can decide whether product requested by employee will come from stock/warehouse directly or it needs to be purchase from vendor. So we have field on requisition lines where responsible can select

Installation

- Copy material_internal_requisition module to addons folder

- Install the module normally like other modules

Internal Requisitions User Access
